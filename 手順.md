# Event Manager B コピペ手順

## 1. ZIPを展開する

`event-manager-b.zip` を任意のフォルダへ展開します。

## 2. ターミナルを開く

PowerShell または Windows Terminal を開き、展開先へ移動します。

```powershell
cd <展開したフォルダ>\event-manager-b
```

## 3. 依存関係をインストールする

```powershell
npm install
```

## 4. 開発サーバーを起動する

```powershell
npm run dev
```

表示された URL を開きます。通常は次です。

```text
http://localhost:5173
```

## 5. デモアカウントでログインする

- 管理者: `admin / admin123`
- 一般ユーザー: `user1 / user123`
- 一般ユーザー: `user2 / user123`

## 6. GitHub Pages に公開する

### 6-1. GitHub に新規リポジトリを作る

例: `event-manager-b`

### 6-2. PowerShell で push する

```powershell
git init
git add .
git commit -m "Initial commit for Event Manager B"
git branch -M main
git remote add origin https://github.com/<GitHubユーザー名>/event-manager-b.git
git push -u origin main
```

### 6-3. GitHub Pages を有効にする

- Settings
- Pages
- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`
- Save

### 6-4. 公開URL

```text
https://<GitHubユーザー名>.github.io/event-manager-b/
```

## 7. 本番ビルド確認

```powershell
npm run build
npm run preview
```
